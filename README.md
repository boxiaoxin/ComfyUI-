ComfyUI功能管理器

<img width="960" height="640" alt="Screenshot_20260305130654" src="https://github.com/user-attachments/assets/5e4a2af9-dd16-41f1-a85b-47fd5835fcdc" />

1.点击silent_launcher.vbs启动

2.首次启动自动检测系统硬件。

<img width="960" height="640" alt="Screenshot_20260305130707" src="https://github.com/user-attachments/assets/4aad585d-ae9d-4b2f-8e3d-81062cf540aa" />

3.选择安装路径后保存，点击开始安装，等待全部安装完毕后会自动测试启动comfyui，能顺利进入浏览器界面说明安装成功（建议路径选择在非系统盘）。

4.安装的comfyui是在虚拟环境中运行。

<img width="960" height="640" alt="Screenshot_20260305130716" src="https://github.com/user-attachments/assets/71729a26-cc6e-4b61-8da8-68402e80047e" />

5.已经内置python和git,不需要依赖系统安装这两个软件。

6.如果comfyui启动错误，请优先使用一键修复。

7.ComfyUI-Manager已经内置安装，其他插件请在管理器中依需下载使用。

<img width="960" height="640" alt="Screenshot_20260305130728" src="https://github.com/user-attachments/assets/64371c42-ee7e-4251-a9c9-59e8079469a0" />

8.工作流、模型文件、个人作品打开方式为双击文件夹。

<img width="960" height="640" alt="Screenshot_20260305130754" src="https://github.com/user-attachments/assets/2d7cafa1-6acf-45f5-b864-09674f985ed3" />

9.一键启动中的自动运行功能为运行程序自动启动comfyui，可以随时关闭。

<img width="960" height="640" alt="Screenshot_20260305130815" src="https://github.com/user-attachments/assets/222ba829-d2fe-4c25-a89b-f81c8e97cf09" />

10.只在intel独显A770测试过软件，一定还存在不少的错误，因此造成的系统崩溃、数据损失等不可预料的事件与作者无关。
